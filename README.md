# Node Api Starter

> Backend api starter build with nodejs

## Developing

A nodejs >= 8.0.0

```bash
# install dependencies
npm install

# serve with hot reload at localhost:3000
npm run start

# build for production
npm start build

# run tests
npm start test
```

## Connecting Database

- [Mongodb](https://www.mongodb.com/)

## Includes
